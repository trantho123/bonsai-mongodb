1. Run Backend
Open the first terminal and navigate to the api folder:
bash
        cd api
        go run main.go
1.1 Run Backend using Docker (Optional)
Alternatively, you can run the backend in a Docker container:
bash
        docker-compose up
2. Run Frontend
Open the second terminal and start the frontend:
bash
        Add .env
        npm start
