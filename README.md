# **How to Run the Project**

---

## **1. Run Backend**
Open the **first terminal** and navigate to the `api` folder:

```bash
cd api
go run main.go
```
Or run Backend using Docker (Optional)
Alternatively, you can run the backend in a Docker container:
```bash
docker-compose up
```
